# Daytona ClickUp Example

This example starts a Daytona sandbox, uploads `sdk/clickup.ts`, and fetches the
ClickUp task `86aeggpdk`.

## Prerequisites

- A `.env` file in this folder with `CLICKUP_TOKEN` set.
- `DAYTONA_API_KEY` and `DAYTONA_BASE_URL` are optional but supported.

## Run

```bash
cd example
npm install
npm run start
```
